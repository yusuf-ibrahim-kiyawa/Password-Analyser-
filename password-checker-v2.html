About & Privacy

How this tool works and what we do with your data.

100% local. Your password is never sent anywhere. All strength analysis and entropy calculation runs entirely in your browser — no network requests, no server, no logging.

How entropy works. Entropy is measured in bits and reflects how many possible combinations an attacker must try. We calculate it from password length × log₂(charset size). 50+ bits is solid; 70+ is excellent.

Generated passwords. Passwords you generate here are created using Math.random(), which is sufficient for most uses but is not cryptographically secure. For high-stakes secrets, use a dedicated password manager.
